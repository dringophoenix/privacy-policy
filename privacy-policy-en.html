Application Name: Fireside Tracker

Thank you for using "Hearthstone Deck Tracker" (hereinafter referred to as "this application"). We highly value user privacy and data security. To ensure you understand how we handle matters related to your information, please read the following content carefully. This privacy policy applies to all services provided by this application, including deck information queries and card tracking features.

I. Information Collection

Non-Personal Information Data

This application may collect the following non-personally identifiable information to optimize user experience and provide services:

  • Device information: Such as device model, operating system version, for adapting application features.
  • Screen content recognition results: The card tracking feature uses local screen text recognition technology to track cards in matches, which is only processed locally on the device and not uploaded to any server.
  • Operation logs: Such as feature usage records, for troubleshooting and service improvement.

Sensitive Information Not Collected

This application does not collect any personally identifiable information (such as name, address, phone number) or personal sensitive data (such as account passwords). We also do not access or store game client data, server information, or other content directly related to user identity.

II. Data Processing

Local Processing

All data processing in this application is completed locally on your device. For example, the card tracking feature generates card statistics through screen text recognition, a process that does not involve data uploading or remote storage.

No Third-Party Analysis

We do not use any third-party analysis tools to track user behavior, nor do we record detailed logs of user operations for analysis or sharing.

III. Security Measures

Although this application does not collect personally identifiable information, we still take the following measures to protect the security of the application and the user's device:

  • Using encryption technology to protect the integrity of in-app data;
  • Regularly updating the application to fix potential security vulnerabilities;
  • Ensuring that the card tracking feature is limited to screen text recognition and does not involve any interaction with the game client or server.

IV. Data Use and Sharing

Purpose of Use

The non-personal information collected by this application is only used to provide services (such as card statistics) and optimize application performance, not for other purposes.

No Data Sharing

Unless required by laws and regulations or explicitly authorized by you, this application will not transfer any data (including device information or recognition results) to third parties.

V. Privacy Policy Changes

If this privacy policy needs to be changed due to legal requirements or functional adjustments, we will inform you through pop-ups or notifications within the application and update the relevant content on this page. The new policy will take effect from the date of publication, and your continued use of this application indicates acceptance of the updated privacy policy.
